### 📊 Predicting Sales Based on Advertising Data

------------------------------------------------------------------------

## ✨ Project Overview

This project analyzes how advertising budgets (TV, Radio, Newspaper) impact product sales using **Machine Learning (Linear Regression)**.

The goal is to build a predictive model that helps businesses optimize marketing strategies and forecast future sales.

### 🎯 Objectives

- ✅ Analyze advertising vs sales relationships
- ✅ Perform data cleaning and preprocessing
- ✅ Conduct exploratory data analysis (EDA)
- ✅ Build regression model for prediction
- ✅ Forecast future sales
- ✅ Generate actionable business insights

------------------------------------------------------------------------

## 📈 Analysis Performed

### 🔹 Data Cleaning

### 🔹 Feature Engineering
- Understanding advertising channels

### 🔹 Exploratory Data Analysis
- Correlation analysis

### 📊 Visualizations Included

- Pairplot analysis
- TV vs Sales regression plot
- Radio vs Sales regression plot
- Newspaper vs Sales regression plot
- Actual vs Predicted sales plot
- Feature importance (coefficients)

### 🔹 Machine Learning Model
- Linear Regression
- Train-test split
- Feature scaling
- Model evaluation (MAE, RMSE, R²)

### 🔹 Business Insights
- TV advertising has highest impact on sales
- Radio has moderate influence
- Newspaper has weakest contribution
- Budget optimization improves ROI
